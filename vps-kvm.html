FAQ

Foire aux questions

Quel est le processeur de vos offres VPS Game KVM ?

Le processeur de nos offres VPS Game KVM est un Intel core i9-9900k.

Puis-je installer docker sur ces VPS ?

L'installation de docker est belle est bien possible.

Puis-je installer Windows Server sur les VPS Game KVM ?

Il n'est pas possible d'installer Windows Server sur ces offres VPS.

Puis-je changer d'offre à tout moment sans aucune perte de données ?

Oui et non, vous pouvez upgrade votre offre en payant la différence sans aucune perte de données, en revanche la technologie KVM ne permet pas de réduire la capacité de disque d'un VPS existant. La rétrogradation des offres VPS Game KVM n'est donc pas possible.

Quelle est la localisation des VPS Game KVM ?

Nos offres VPS Game KVM se situent sur le continent européen.
